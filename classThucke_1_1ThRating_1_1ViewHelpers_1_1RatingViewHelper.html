Detailed Description

The Rating Viewhelper

Renders the rating view based upon plugin.tx_thrating Only the argument ratedobjectuid is required. Others could be used to configure the output

= Example =

<thr:rating ratetable="some_tablename" ratefield="one_field_of_the_table" ratedobjectuid="UID integer" ></thr:rating> <output> rendered rating </output>
